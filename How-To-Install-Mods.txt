########################################
Installing Mods for Monster Girl Dreams
########################################

Below you will find instructions on how to install mods for all desktop platforms.

Documentation on how to make mods can be found in ModdingDocs/Homepage.html. That will open offline documentation in your browser.
Alternatively, you can find it online at https://readthedocs.org/projects/mgd-modding-docs/

You can find mods through the community-maintained list on the wiki at:
https://monstergirldreams.miraheze.org/wiki/Category:List_Of_Mods

And on the MGD Discord in #mod-posting: https://discord.com/invite/Md5n5KJ

###################
How to Install Mods
###################
  Ensure the save you will be loading is in the town square whenever updating the selection of mods!
  Optionally, name the save via the option in the bottom right of the save menu and keep a vanilla and modded save.
  It is unlikely a modded save will become unstable, but it could influence the stats of your character in a undesirable way.

  It is recommended to use the in-game mod menu to download and install mods to avoid human error. Optionally, you can follow the manual steps below. 

  # Windows/Linux
    1. Navigate to Mods/ from the base folder of Monster Girl Dreams, where you found this txt file, and the game executable.
    2. Extract the .zip file for the mod and place its **default mod folder** (which contains multiple subfolders) into Mods/
  # macOS
    1. Right click the MonGirlDreams application, select "Show Package Contents". Navigate to Contents/Resources/autorun/Mods/
    2. Extract the .zip file for the mod and place its **default mod folder** (which contains multiple subfolders) into Mods/
  3. Launch the game!
  
  You can verify it is correct via the in-game mod menu. If it is incorrectly extracted, there will be a green "Fix Path" button you can press to fix it.

#####################
How to Uninstall Mods
#####################
  You have two choices! Disabling a mod, or, actually removing a mod.
  Additionally, your modded save is safe to play after uninstalling a mod, provided you load in from the town square.
  
  NOTE: You can use the in-game mod menu to disable and remove mods as well.
  
  # Disabling Mods
    0. Ensure the save you will be loading is in the town square whenever updating the selection of mods!
    1. Rename the mod in Mods/ to start with an _underscore. Such as "_ModExample"
    * To re-enable, simply rename the mod to not have the _underscore at the front.

  # Removing Mods
    0. Ensure the save you will be loading is in the town square whenever updating the selection of mods!
    1. Simply delete the mod in Mods/
